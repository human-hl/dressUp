Веб-приложение для систематизации личного гардероба, создания образов с помощью drag-and-drop и получения стилистических советов.

- React 18 + TypeScript
- Redux Toolkit
- Material-UI (MUI)
- @dnd-kit (drag-and-drop)
- Vite

- NestJS + TypeScript
- TypeORM + MySQL
- JWT-авторизация
- Multer (загрузка файлов)
- @imgly/background-removal-node (удаление фона, нейросеть U-Net)

## Установка и запуск

### Бэкенд
```bash
cd backend
npm install
npm run start:dev
```

### Фронтенд
```bash
cd frontend
npm install
npm run dev
```

###  База данных
Установи MySQL
Создай базу dressup
TypeORM создаст таблицы автоматически (synchronize: true)
